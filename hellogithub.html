С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения. Git помогает вернуть файлы в исходное состояние и видеть изменения, внесённые в определённый период. Разработчики выполняют разные команды (например, commit, push), а все изменения синхранизируются с центарльным репозиторием. Git - это система контроля версий, а GitHub - онлайн-сервис, по сути социальная цеть. Одна из основных целей GitHub - быть единым местом для проектов с исходным кодом. Предпологается, что пользователь делится чем-то полезным, а другие люди смогут участвовать в разработке.
